# La-scimmia-cifratrice
Questo progetto, sviluppato come elaborato finale per l'esame di stato 2020-2021, unisce il Test di Turing e il teorema della scimmia instancabile.

Un'interfaccia software permette di cifrare una frase e salvarla in un database. Contemporaneamente, un dispositivo hardware ("la scimmia") connesso via porta seriale genera una sequenza casuale di simboli della stessa lunghezza. Comparando i risultati del dispositivo scimmia alle frasi criptate, è indistinguibile quale dei due origini da una frase scritta da un essere umano.

Il circuito è stato progettato in TinkerCad prima della realizzazione fisica. Nella repository è disponibile il file .brd con lo schema completo.

Tecnologie: C# (WinForms), Arduino, Database SQL, PHP, HTML, CSS.
